# admin-cms
React app for manage data from express-cms in other repository

[![img-capture-2]][url-capture-2]

## Installation

```sh
git clone https://github.com/muhammadardie/admin-cms.git
cd admin-cms
npm install
cp .env.example .env
npm start
```
